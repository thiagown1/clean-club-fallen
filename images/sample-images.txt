- Aspect ratio: 1:1 (square) recommended

## Instructions
1. Place your images in this `/images/` directory
2. Update the HTML file to uncomment the image tags
3. Test the page to ensure images load correctly
4. The page will automatically fall back to mockups if images fail to load
